# VocaDB Web Frontend - ASP.NET Core

MVC views and Web APIs.

See [VocaDB development environment](https://github.com/VocaDB/vocadb/wiki/VocaDB-development-environment).

## Environment

* .NET Core SDK
* node.js with NPM (latest LTS)

## Compiling frontend assets

On first run, install dependencies with ```npm install```.

To compile development build, run ```npm run dev```.
